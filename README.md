# Quickstart

```shell
asdf plugin add gcloud
```

# Pre-requisits

- Python 2 or Python 3
- set `CLOUDSDK_PYTHON` environment variable in your shell config to load the correct version of Python.

# Install

```shell
asdf plugin add gcloud
# or
asdf plugin add https://github.com/jthegedus/asdf-gcloud.git
```
